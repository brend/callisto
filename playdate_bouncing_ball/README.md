# Bouncing Ball (Playdate + Callisto)

A tiny Playdate sample using the Callisto workflow from `docs/playdate_workflow.md`.
It follows the documented `Source/main.lua` shim pattern: Lua owns frame-to-frame state, while gameplay logic is implemented in Callisto.
Turn the crank to change vertical velocity and curve the ball's trajectory.

## Layout

- `src/game.cal`: game logic in Callisto
- `Source/main.lua`: Playdate shim that sets `playdate.update`
- `Source/game.lua`: generated by Callisto
- `BouncingBall.pdx`: built by `pdc`

## Build

From this directory:

```sh
make build-lua
```

To build the `.pdx` (requires Playdate SDK `pdc`):

```sh
make build
```

To open in simulator:

```sh
make run
```
