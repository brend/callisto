# Bouncing Ball (Playdate + Callisto)

A tiny Playdate sample using the Callisto workflow from `docs/playdate_workflow.md`.
It follows the documented `Source/main.lua` shim pattern: Lua owns frame-to-frame state, while gameplay logic is implemented in Callisto.
Use crank or d-pad control to curve the ball trajectory.

## Layout

- `src/game.cal`: game logic in Callisto
- `../playdate_bindings/src/playdate/*.cal`: shared Playdate extern bindings
- `Source/main.lua`: Playdate shim that sets `playdate.update`
- `Source/game.lua`: generated by Callisto
- `Source/playdate.lua`: generated root Playdate extern module
- `Source/playdate/graphics.lua`: generated module for shared extern bindings
- `Source/playdate/input.lua`: generated input helper module
- `BouncingBall.pdx`: built by `pdc`

## Controls

- Crank: curve vertically in default mode
- Hold `A`: switch to d-pad mode (`Up`/`Down`)
- Press `B`: reset ball to center

`Source/main.lua` also preloads `playdate/input` so wrapper helpers are available at `playdate.input.*`.

## Build

From this directory:

```sh
make build-lua
```

To build the `.pdx` (requires Playdate SDK `pdc`):

```sh
make build
```

To open in simulator:

```sh
make run
```
